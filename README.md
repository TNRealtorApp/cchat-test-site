# cchat-test-site

A whimsical Tennessee Time Travel landing page, built end-to-end through cchat
as the first real test of the multi-Claude coordination system.

This repo deliberately doesn't matter — it's a sandbox for shaking out cchat
before any production work. If it looks weird, that's the point.
